POSTGRESSQL Setup
1. Install postgresql -> https://www.postgresql.org/download/
2. Change the password of the default postgresql username:
sudo -u postgres psql
ALTER USER postgres WITH PASSWORD 'newpassword';
- Ensure the result is: ALTER ROLE
- Place that new password in the .env file
3. Start the database: sudo service postgresql start
4. Note down the port when using: sudo service postgresql status, place the port in the .env file

5. Note: You can shut down the postgresql with sudo service postgresql stop
